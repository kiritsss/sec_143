HOW TO PLAY
1. A number falls in one of the 3 columns.
Un número cae en una de las 3 columnas.
2. Type the number in English and press ENTER.
Escribe el número en inglés y presiona ENTER.
3. Correct answer = the block disappears!
Respuesta correcta = ¡el bloque desaparece!
4. Wrong answer = the block keeps falling.
Respuesta incorrecta = el bloque sigue cayendo.
5. If a block reaches the bottom, it stacks up.
Si el bloque llega al fondo, se apila.
6. Don't let any column reach the top!
¡No dejes que ninguna columna llegue arriba!
